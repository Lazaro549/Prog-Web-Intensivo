An array, or structure, is a collection of variables. It's like a list or a drawer with compartments.
To access these variables, you use an index or position.

Important: positions in arrays start with the number zero.

**Declaration**:

`$valuble=array();`

**New value**:

`$valuble[]="New Value";`

**Modificar**:

`$valuble[<index>]="New Value";`

**Delete**:
`unset($valuble[<index>]);`

`<index>`: array position

**Accessories**:

Display an array for debugging (Do not use in production)

`echo "<pre>";`

`print_r($Valuble);`

`echo "</pre>";`
